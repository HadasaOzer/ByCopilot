About Dark Mode Toggle

This demo implements a dark mode toggle feature with:

  • Accessibility: Keyboard navigable and screen reader compatible
  • Persistence: Your theme choice is saved to localStorage
  • Smooth Transitions: Theme changes with 0.3s ease animation
  • Cross-browser Support: Works on Chrome, Firefox, Safari, and Edge
